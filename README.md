# 당뇨살림

당뇨를 겪는 사람과 가족을 위한 정적 웹사이트입니다.

## 리뉴얼 콘셉트

- 환자의 몸으로 겪는 불안
- 오랜 진료실에서 본 반복되는 실수
- 초기 환자부터 합병증을 겪는 환자까지의 단계별 이야기
- 혈당, 식사, 약, 합병증, 가족, 병원 선택을 생활 중심으로 정리

## Vercel 배포

이 저장소가 Vercel 프로젝트와 연결되어 있으면 main 브랜치에 커밋될 때 자동 배포됩니다.

## 주의

이 사이트는 의료기관이 아니며 진단, 처방, 약물 변경을 안내하지 않습니다. 응급 상황이나 치료 변경은 담당 의료진과 상의해야 합니다.
